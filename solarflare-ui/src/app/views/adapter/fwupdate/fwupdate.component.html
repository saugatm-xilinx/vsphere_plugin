Help

The firmware update will program (upgrade or downgrade) a firmware image on the adapter. The firmware image can be any of the following:
  • The available firmware version - packaged along with the plugin installer.
  • A custom version installed from a firmware image (.dat) file copied to the host on which browser is running.
  • A custom version installed from a URL.
Available Firmware:
Select the UPDATE button to update firmware to version packaged within the plugin installer.
Custom Update:
Select the CUSTOM UPDATE button. Firmware can be installed from a file on the local host or from a remote server identified by URL.
NOTE:
Firmware can also be upgraded using the sfupdate_esxi utility from a remote host. sfupdate_esxi connects to the CIM Provider on the local ESXi host. For more information about the CIM Provider and sfupdate_esxi utility, please refer to the Solarflare Server User Guide (SF-103837-CD, VMware section).
