			
Windows Installer requires a Java Standard Edition Runtime
Environment (JRE) version 8 or later.

=============================
Running With JRE 8 Or Later
=============================

(1) Download and Install a Java SE Runtime Environment (JRE)

(1.1) Download a Java SE Runtime Environment (JRE),
      release version 8 or later, from
      http://www.oracle.com/technetwork/java/javase/downloads/index.html

(1.2) Install the JRE according to the instructions included with the
      release.

      You may also use a full Java Development Kit (JDK) rather than just
      a JRE.
	  

============================
Building Windows Installer
============================
NOTE: Build only when you have any changes to VCP plugin. Otherwise just run the installer.
Follow the "BuildInformation_Windows MSI_Installer.doc" file in "\installer\Windows\doc" directory.

============================
Running Windows Installer
============================

Run .msi created in "\installer\Windows\build" folder.
